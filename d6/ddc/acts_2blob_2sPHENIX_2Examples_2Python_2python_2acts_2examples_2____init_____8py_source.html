142 def NamedTypeArgs(**namedTypeArgs):
143  """Decorator to move args of a named type (e.g. `namedtuple` or `Enum`) to kwargs based on type, so user doesn't need to specify the key name.
144  Also allows the keyword argument to be converted from a built-in type (eg. `tuple` or `int`).
