7 // Comprehensive information about the database as a whole.
8 //
9 // Many of the methods here return lists of information in the form
10 // of TSQLResultSet objects. You can use the normal TSQLResultSet
11 // methods such as TSQLResultSet::GetString() and
12 // TSQLResultSet::GetInt() to retrieve the data from
13 // these TSQLResultSet s. If a given form of metadata is not available,
14 // these methods should throw an TSQLException.
15 //
16 // Some of these methods take arguments that are string patterns.
17 // These arguments all have names such as fooPattern. Within a
18 // pattern string, "%" means match any substring of 0 or more
19 // characters, and "_" means match any one character. Only metadata
20 // entries matching the search pattern are returned. If a search
21 // pattern argument is set to a null ref, that argument's criteria
22 // will be dropped from the search.
