Detailed Description

Material decorator using a map as input.

This reads in map with binning information to decorate the detector with proto-material for material mapping. This allows us to change the mapping parameters directly in the C++ code. Takes a tracking geometry in input, all the surface with mapMaterial=true will be added to a binning map.
